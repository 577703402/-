    Kubernetes集群可用性验证

    Kubernetes k8s
    发布时间 : 2024-03-04 23:05
    阅读 :
    • 节点是否正常

      $ kubectl get node

    • Pod是否正常

      $ kubectl get pod -A

    • 集群网段是否存在冲突
      1. $ kubectl get svc
      2. $ kubectl get pod -A -o wide
    • 是否可以正常创建Pod资源

      $ kubectl create deploy cluster-test –image=registry.cn-beijing.aliyuncs.com/dotbalo/debug-toos – sleep 3600

    • Pod是否能够解析Service（同namespace和跨namespace）
      1. $ nslookup kubernetes
      2. $ nslookup kube-dns.kube-system
    • 每个节点是否能够访问Kubernetes的kubernetes svc 443和kube-dns的service 53端口
    • Pod和Pod之间是否能够正常通讯（同namespace和跨namespace）
    • Pod和Pod之间是否能够正常通讯（同机器和跨机器）
