    Linux 系统介绍

    1. 组成说明
      Linux操作系统由linux内核和各种外围程序组成。

      内核是一个特殊的软件程序，用于实现CPU和内存分配，进程调度，设备驱动等核心操作，以面向硬件为主。
      外围程序包括分析用户指令的解释器，网络服务程序，图形桌面程序等各种应用型的软件程序，以面向用户为主。
      内核的主要功能：① 识别硬件并驱使硬件工作，② 为外围程序提供运行平台。

    2. Shell
      就是一个命令行工具。

      shell也称为终端或壳充当的是人与内核（硬件）之间的翻译官，用户把一些命令告诉终端，他就会调用相对应的程序服务去完成某些工作，主流Linux系统选择Bash解释器作为命令行终端。

    3. 内核说明
      Linux内核的官方网站是http：//www.kernel.org/

      Linux内核最初由芬兰大学许李纳斯·托沃兹（Linus Torvalds）在1991年发布，主要是使用C语言及一小部分汇编语言开发而成。
      Linux内核的标志是一个名为Tux的小企鹅；1994年3月，Linux1.0版本发布。

    4. 外围程序说明
      构成Linux系统的外围程序大部分来自于GNU项目或其他组织的开源软件。

      如：C语言编译工具gcc，命令解释器程序bash，网站服务器程序httpd等。
      GNU名称来源于GNU is not unix的缩写，由自由软件运动的倡导者Richard stallman与1984年开始发起并创建，其目标是编写大量兼容于UNIX操作系统的可自由传播，使用的软件用于替换UNIX系统中的各种商业软件。
      GNU项目的官方网站位于：http：//www.gnu.org/

    系统启动流程

    设备开机按钮 -> 主板供电 -> bios加点自检 -> 检测硬件 -> 根据bios决定启动介质 -> 寻找磁盘上的MBR；加载bootloader -> 内核启动init -> 系统初始化 -> /etc/inittab判断启动级别 -> 读取/etc/rc.d/rcX.d下的脚本文件，运行各种应用软件 -> /etc/rc.local文件，进入到登录界面 -> 输入用户账号和密码 -> /etc/profile。
    其实在运行内核后可以简单理解为：启动init进程，做初始化，读取各种文件，开启进程；bios会检测硬件，是一组固化到计算机主板上的一个ROM芯片上的程序，保存着最重要的基本输入输出的程序、系统设置信息、开机后自检程序和系统自启动程序；主要功能是为计算机提供最底层的、最直接的硬件设置和控制。MBR：主引导记录，大小512字节；bootloader引导加载程序，存储的是开机管理程序，对于Linux来说就是grub程序；主要功能识别操作系统的文件系统格式及加载内核和驱动；bootloader其实是分为两个阶段完成的。
